# Jason Docs

Welcome to your new MkDocs + Material site.

This repository is now set up so you can publish documentation and blog-style articles from Markdown files.

## Quick start

1. Create or edit Markdown files in the docs directory.
2. Update navigation in mkdocs.yml.
3. Push to master and GitHub Actions will publish the site.

## Suggested article layout

- docs/articles/my-topic.md
- docs/articles/release-notes-2026-06.md
- docs/articles/how-i-built-x.md
